MaidSafe-CRUX
=============

Work-in-progress.

CRUX (Connected Reliable Udp eXchange) is a reliable UDP protocol.

Maidsafe-CRUX is an implementation of CRUX for [Boost.Asio](http://www.boost.org/doc/libs/release/libs/asio/).
